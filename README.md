# CutURL

CutURL is a simple web application that shortens URLs for easy sharing.

## Features

- URL Shortening
- Copy to Clipboard


## Usage

1. Enter a long URL.
2. Click "Shorten" to get a shortened URL.
3. Click "Copy to Clipboard" to copy it.

## Installation

Visit [CutURL.vercel.app](https://cuturl.vercel.app).
